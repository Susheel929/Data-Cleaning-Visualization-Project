# 🏪 Real-World Data Project (Retail Domain)

A complete end-to-end data science project on retail sales data
covering data analysis, business insights, ML prediction,
and a professional analytics dashboard.

## 🎯 Objective
Work on a domain-specific dataset (Retail) for applied learning —
perform end-to-end data analysis and prediction, present
findings with visualizations and conclusions.

## ✨ Key Features
- Retail superstore sales dataset (1500 records)
- Business analysis (store, category, city, season)
- 4 ML models to predict revenue
- 14 professional visualizations + dashboard
- Structured conclusions report

## 🛠️ Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Scikit-learn
